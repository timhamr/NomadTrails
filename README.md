# Nomad Trails Campervan Chat

Static prototype of the Nomad Trails campervan rental assistant. Deployed via GitHub Pages with `index.html` plus supporting images.

## Running Locally

Open `index.html` in any modern browser.

## Deploying to GitHub Pages

1. Push this repository to GitHub.
2. In repository settings, enable GitHub Pages for the `main` branch, root (`/`) directory.
3. Visit the generated Pages URL once deployment finishes.
